# News card

## Introduction

A stats card is used to display statistics.

## When to use

A stats card is used in a group in columns, or on its own.

## Alternatives and related components

- Columns

## Anatomy

A stats card consist of:

1. Card
2. Icon
3. Number
4. Description

## (Interactive) states

The stats card contains no interactive states.

## Design properties

### Typography

- Number: Mix/2xl/700
- Description: Sans/md/400

## Colors

- Text: Green/5
- Background: grey/2

Be concise in the description length. The description should be no longer than x characters.

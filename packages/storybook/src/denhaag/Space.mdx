# Space

Use the design tokens for space only for space within (inline/block) or between components (column/row), not to define the size of elements.

## Inline spacing

Inside components, use inline spacing before and after text. In CSS you can use the following properties:

- `margin-inline-start`
- `margin-inline-end`
- `padding-inline-start`
- `padding-inline-end`

<SpaceTokensTable orientation="inline" tokens={Object.values(designTokens["denhaag"]["space"]["inline"])} />

## Block spacing

Inside components, use block spacing above and below text. In CSS you can use the following properties:

- `margin-block-start`
- `margin-block-end`
- `padding-block-start`
- `padding-block-end`
